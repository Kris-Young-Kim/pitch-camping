# 비즈니스 모델 문서 (Pitch Camping)

## 1. 서비스 가치 제안

### 1.1 핵심 가치
- **통합 정보 플랫폼**: 전국 캠핑장 정보를 한 곳에서 검색·비교·선택
- **시각적 정보 제공**: 네이버 지도 연동으로 위치 기반 직관적 정보 제공
- **안전한 캠핑 문화**: 안전 수칙 정보 제공으로 사용자 안전 보장
- **개인화된 추천**: 북마크, 리뷰, 통계 기반 맞춤형 캠핑장 추천

### 1.2 사용자 이점
- **시간 절약**: 여러 사이트를 돌아다니며 정보를 찾지 않아도 됨
- **신뢰성**: 공공 데이터(고캠핑 API) 기반 정확한 정보 제공
- **편의성**: 모바일 최적화로 언제 어디서나 접근 가능
- **커뮤니티**: 리뷰와 북마크 기능으로 사용자 간 정보 공유

## 2. 타겟 시장 분석

### 2.1 타겟 사용자
- **주 타겟**: 20-40대 캠핑 애호가
  - 캠핑을 즐기는 개인/가족
  - 신규 캠핑 입문자
  - 캠핑 장비 구매 고려자
- **부 타겟**: 캠핑 관련 사업자
  - 캠핑장 운영사
  - 캠핑 장비 판매업체
  - 캠핑 투어/패키지 업체

### 2.2 시장 규모
- **국내 캠핑 시장**: 연간 약 2조원 규모 (2023년 기준)
- **캠핑 인구 증가 추세**: 코로나 이후 지속적인 성장
- **전국 캠핑장 수**: 약 1,500개 이상 (고캠핑 API 기준)
- **디지털 접근성**: 모바일 인터넷 사용률 95% 이상

### 2.3 시장 트렌드
- **글램핑/카라반 인기**: 편의성 추구하는 캠핑 문화 확산
- **SNS 공유 문화**: 캠핑 경험 공유를 통한 확산
- **지속가능한 관광**: 친환경 캠핑 관심 증가
- **가족 단위 여행**: 가족 단위 캠핑 증가

## 3. 수익 모델

### 3.1 수익원 (Revenue Streams)

#### 3.1.1 광고 수익 (초기 주요 수익원)
- **배너 광고**: 캠핑장 전체 목록 페이지
- **스폰서 광고**: 상위 검색 결과 노출
- **캠핑장 프로모션**: 특정 캠핑장 상세페이지 주요 배치
- **예상 수익**: 월 10만-100만원 (초기), 연간 1,000만-5,000만원 (성장 단계)

#### 3.1.2 예약 수수료 (중장기 수익원)
- **예약 시스템 연동**: 외부 예약 플랫폼과의 제휴
- **수수료율**: 예약 금액의 5-10%
- **예상 수익**: 월 50만-500만원 (성장 단계), 연간 1억-5억원 (확장 단계)

#### 3.1.3 프리미엄 구독 (장기 수익원)
- **프리미엄 기능**: 
  - 광고 없는 경험
  - 우선 예약 알림
  - 고급 필터링 옵션
  - 무제한 북마크
  - 독점 혜택 및 할인
- **가격**: 월 4,900원 또는 연 49,000원
- **예상 수익**: 월 50만-500만원 (성장 단계), 연간 1억-5억원 (확장 단계)

#### 3.1.4 제휴 마케팅 (보조 수익원)
- **캠핑 장비 제휴**: 온라인 쇼핑몰 제휴 링크
- **캠핑 투어 제휴**: 패키지 상품 판매 수수료
- **예상 수익**: 월 10만-100만원 (초기), 연간 500만-5,000만원 (성장 단계)

#### 3.1.5 데이터 라이선싱 (장기 수익원)
- **캠핑장 통계 데이터**: 관광 관련 기관/기업에 제공
- **사용자 인사이트**: 마케팅 리서치 데이터 제공
- **예상 수익**: 연간 1,000만-5,000만원 (확장 단계)

### 3.2 수익 모델 우선순위
1. **Phase 1 (초기)**: 광고 수익 중심
2. **Phase 2 (성장)**: 예약 수수료 + 광고 수익 병행
3. **Phase 3 (확장)**: 구독 수익 + 예약 수수료 + 광고 수익
4. **Phase 4 (성숙)**: 데이터 라이선싱 추가

## 4. 경쟁 분석

### 4.1 경쟁사
- **고캠핑 공식 사이트**: 공공 데이터 중심, 사용자 경험 부족
- **캠핑 관련 블로그/커뮤니티**: 정보 분산, 검색 어려움
- **구글/네이버 지도**: 기본 정보만 제공, 캠핑 특화 기능 부족
- **캠핑 예약 플랫폼**: 예약 위주, 정보 검색 기능 약함

### 4.2 경쟁 우위
- **통합 정보 제공**: 검색, 지도, 상세 정보를 한 곳에서 제공
- **사용자 중심 UX**: 현대적이고 직관적인 사용자 인터페이스
- **확장 가능한 아키텍처**: Next.js 기반 빠른 성능과 확장성
- **데이터 기반 추천**: 통계 및 리뷰 기반 맞춤형 추천
- **안전 수칙 정보**: 안전한 캠핑 문화 확산을 위한 차별화 기능

### 4.3 경쟁 전략
- **차별화 포인트**: 사용자 경험, 안전 정보, 커뮤니티 기능
- **파트너십**: 캠핑장 운영사, 관광청, 캠핑 관련 협회와의 협력
- **기술적 우위**: 최신 웹 기술 스택 활용, 빠른 개발 속도
- **사용자 커뮤니티**: 리뷰, 북마크, 공유 기능을 통한 사용자 참여

## 5. 성장 전략

### 5.1 초기 성장 전략 (0-6개월)
- **내용 마케팅**: 캠핑 관련 블로그 포스팅, SEO 최적화
- **소셜 미디어**: 인스타그램, 페이스북을 통한 서비스 홍보
- **파트너십**: 캠핑 블로거, 유튜버와의 협업
- **사용자 피드백**: 피드백 수집 및 빠른 개선 사이클

### 5.2 성장 단계 전략 (6-12개월)
- **기능 확장**: 날씨 정보, 교통 정보, 실시간 예약 가능 여부
- **프리미엄 기능**: 구독 서비스 런칭
- **모바일 앱**: 네이티브 앱 개발 고려
- **B2B 확장**: 캠핑장 운영사 대상 마케팅 솔루션 제공

### 5.3 확장 단계 전략 (12개월 이상)
- **데이터 라이선싱**: 통계 및 인사이트 데이터 판매
- **국제화**: 해외 캠핑 정보 확장 (선택적)
- **에코시스템 구축**: 캠핑 장비 구매, 캠핑 투어 패키지 연계
- **IPO/인수**: 장기적 사업 모델 확립 후 고려

## 6. 주요 지표 (KPIs)

### 6.1 사용자 지표
- **DAU (일일 활성 사용자)**: 목표 1,000명 (6개월), 10,000명 (12개월)
- **WAU (주간 활성 사용자)**: 목표 5,000명 (6개월), 50,000명 (12개월)
- **MAU (월간 활성 사용자)**: 목표 20,000명 (6개월), 200,000명 (12개월)
- **사용자 유지율**: 30일 유지율 40% 이상

### 6.2 수익 지표
- **월간 수익 (MRR)**: 목표 100만원 (6개월), 1,000만원 (12개월)
- **연간 수익 (ARR)**: 목표 1,200만원 (6개월), 1억 2천만원 (12개월)
- **고객 생애 가치 (LTV)**: 목표 10,000원 이상
- **고객 획득 비용 (CAC)**: 목표 5,000원 이하

### 6.3 참여 지표
- **페이지 뷰**: 월평균 100,000회 (6개월), 1,000,000회 (12개월)
- **북마크 수**: 월평균 10,000개 (6개월), 100,000개 (12개월)
- **리뷰 작성 수**: 월평균 500개 (6개월), 5,000개 (12개월)
- **평균 세션 시간**: 5분 이상

### 6.4 기술 지표
- **페이지 로드 시간**: 2초 이하
- **API 응답 시간**: 1초 이하
- **에러율**: 0.1% 이하
- **Lighthouse 점수**: 80점 이상

## 7. 리스크 및 대응 전략

### 7.1 기술적 리스크
- **API 의존성**: 고캠핑 API 장애 시 서비스 중단
  - **대응**: 데이터 캐싱 강화, 다중 데이터 소스 준비
- **트래픽 폭증**: 급격한 사용자 증가 시 서버 부하
  - **대응**: 자동 스케일링, CDN 활용, 성능 최적화

### 7.2 사업적 리스크
- **경쟁사 등장**: 대형 플랫폼의 캠핑 서비스 진입
  - **대응**: 사용자 경험 차별화, 빠른 기능 개선, 커뮤니티 강화
- **수익 확보 지연**: 광고 수익이 예상보다 낮을 경우
  - **대응**: 다각화된 수익 구조, 제휴 마케팅 확대, 구독 서비스 빠른 런칭

### 7.3 법적/규제 리스크
- **데이터 프라이버시**: 개인정보 보호법 준수
  - **대응**: 개인정보 처리방침 명확화, GDPR 준수, 사용자 동의 프로세스
- **광고 규제**: 광고 관련 법규 준수
  - **대응**: 광고 표시 가이드라인 준수, 명시적 광고 표시

## 8. 피드백 반영 계획

### 8.1 사용자 피드백 수집
- **인앱 피드백**: 피드백 폼을 통한 지속적 수집
- **사용자 인터뷰**: 정기적인 사용자 인터뷰 진행
- **데이터 분석**: 사용자 행동 데이터 분석을 통한 인사이트 도출

### 8.2 피드백 처리 프로세스
1. **수집**: 피드백 폼, 이메일, 리뷰를 통한 수집
2. **분류**: 우선순위, 카테고리별 분류
3. **검토**: 주간/월간 피드백 리뷰 미팅
4. **개선**: 개발 로드맵에 반영 및 개발 진행
5. **공지**: 개선 사항 사용자에게 공지

### 8.3 주요 피드백 영역
- **기능 요청**: 새 기능 추가 요청
- **버그 리포트**: 기술적 문제 해결
- **UX 개선**: 사용자 경험 개선 제안
- **데이터 정확도**: 캠핑장 정보 정확도 개선
