## 슬라이드 2: 문제 정의 (Problem)

### 현재 캠핑장 정보 검색의 문제점

- **정보 분산**: 여러 사이트를 돌아다니며 정보를 찾아야 함
- **불일치 정보**: 사이트마다 다른 정보로 혼란 발생
- **비시각적**: 지도 기반 직관적 정보 제공 부족
- **불완전한 정보**: 예약 가능 여부, 실시간 정보 부족
- **안전 정보 부재**: 계절별/주제별 안전 수칙 정보 부족

### 사용자 페인 포인트

- 캠핑 계획에 많은 시간 소요
- 신뢰할 수 있는 정보 부족
- 반려동물 동반, 특정 시설이 필요한 캠핑장 찾기 어려움

---

## 슬라이드 3: 솔루션 (Solution)

### Pitch Camping이 제공하는 솔루션

**✅ 통합 정보 플랫폼**
- 전국 1,500개 이상 캠핑장 정보를 한 곳에서
- 공공 데이터(고캠핑 API) 기반 정확한 정보 제공

**✅ 시각적 정보 제공**
- 네이버 지도 연동으로 위치 기반 직관적 정보
- 이미지 갤러리, 상세 정보 제공

**✅ 스마트 검색 및 필터링**
- 지역, 타입, 시설, 키워드 기반 검색
- 북마크, 리뷰 기반 맞춤형 추천

**✅ 안전한 캠핑 문화 확산**
- 계절별/주제별 안전 수칙 정보 제공

---

## 슬라이드 4: 시장 기회 (Market Opportunity)

### 국내 캠핑 시장 규모

- **연간 시장 규모**: 약 2조원 (2023년 기준)
- **캠핑 인구**: 지속적인 증가 추세 (코로나 이후)
- **전국 캠핑장 수**: 약 1,500개 이상 (고캠핑 API 기준)
- **디지털 접근성**: 모바일 인터넷 사용률 95% 이상

### 시장 트렌드

- **글램핑/카라반 인기**: 편의성 추구하는 캠핑 문화 확산
- **SNS 공유 문화**: 캠핑 경험 공유를 통한 확산
- **가족 단위 여행**: 가족 단위 캠핑 증가
- **지속가능한 관광**: 친환경 캠핑 관심 증가

### 타겟 사용자

- **주 타겟**: 20-40대 캠핑 애호가 (개인/가족)
- **부 타겟**: 캠핑 관련 사업자 (캠핑장 운영사, 장비 판매업체)

## 슬라이드 5: 제품 데모 (Product Demo)

### 핵심 기능

1. **캠핑장 목록 + 필터링**
   - 지역, 타입, 시설 필터
   - 키워드 검색
   - 정렬 옵션

2. **네이버 지도 연동**
   - 지도에서 캠핑장 위치 확인
   - 목록-지도 동기화
   - 마커 클릭으로 상세 정보 확인

3. **상세 정보 페이지**
   - 캠핑장 이미지 갤러리
   - 기본 정보 (주소, 전화번호, 홈페이지)
   - 예약 버튼 (홈페이지/전화)
   - 리뷰 및 평점
   - 안전 수칙 추천

4. **사용자 기능**
   - 북마크 (로그인 사용자)
   - 리뷰 작성 및 평점
   - 공유 기능

### 기술 스택

- **프론트엔드**: Next.js 15, React 19, Tailwind CSS v4
- **백엔드**: Supabase (PostgreSQL)
- **인증**: Clerk
- **지도**: 네이버 지도 API v3
- **UI 컴포넌트**: shadcn/ui

## 슬라이드 6: 비즈니스 모델 (Business Model)

### 수익원 (Revenue Streams)

1. **광고 수익** (초기 주요 수익원)
   - 배너 광고, 스폰서 광고
   - 예상 수익: 연간 1,000만-5,000만원 (성장 단계)

2. **예약 수수료** (중장기 수익원)
   - 예약 시스템 연동 수수료 (5-10%)
   - 예상 수익: 연간 1억-5억원 (확장 단계)

3. **프리미엄 구독** (장기 수익원)
   - 광고 없는 경험, 우선 예약 알림, 고급 필터링
   - 가격: 월 4,900원 또는 연 49,000원
   - 예상 수익: 연간 1억-5억원 (확장 단계)

4. **제휴 마케팅** (보조 수익원)
   - 캠핑 장비/투어 제휴 링크 수수료
   - 예상 수익: 연간 500만-5,000만원 (성장 단계)

5. **데이터 라이선싱** (장기 수익원)
   - 캠핑장 통계 데이터, 사용자 인사이트 제공
   - 예상 수익: 연간 1,000만-5,000만원 (확장 단계)

### 수익 모델 우선순위

- **Phase 1 (초기)**: 광고 수익 중심
- **Phase 2 (성장)**: 예약 수수료 + 광고 수익 병행
- **Phase 3 (확장)**: 구독 수익 + 예약 수수료 + 광고 수익
- **Phase 4 (성숙)**: 데이터 라이선싱 추가
